# UX / Page Structure v0.8

## トップページ

トップページは作業画面ではありません。登録ASIN数、出品候補、出品中、要確認、未返信、今月売上だけを表示し、各メニューへ移動する導線を置きます。

## 商品登録

ASIN / Amazon URL / CSV / 手入力で商品マスタに登録します。SP-API・Keepa・手入力は取得元として保持します。

## 出品候補

全対象国を表示し、ユーザーが国別チェックボックスで出品対象国を選びます。選択された国だけCSV、APIアップロード、自動更新対象になります。

## 出品中商品

出品済み商品の価格・在庫・販売数・最終更新を管理します。

## 販売済み

注文単位で、注文時/発送時の為替固定、発送状況、予測利益を確認します。

## 売上管理

着金後の実利益と予測との差額を確認します。
